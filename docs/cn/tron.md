# Native/Web DAPP 开发

## SimpleWallet 协议

您可以通过麦子钱包定制版 SimpleWallet 协议进行：

1 Native APP 跳转麦子钱包支付或合约签名

2 网页跳转麦子钱包支付或合约签名

协议标准请查看：

[https://github.com/MediShares/SimpleWallet](https://github.com/MediShares/SimpleWallet)

注意：transferReq.blockchain 参数请传 tron
